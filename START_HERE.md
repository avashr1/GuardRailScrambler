## 💡 WHAT IT DOES

### Medical Data Masking
Takes sensitive medical data and replaces:
- ✅ Email addresses → Random valid emails
- ✅ Phone numbers → Random SA format numbers
- ✅ IDs → Random 13-digit strings
- ✅ Medical aid numbers → Random similar format
- ✅ ICD-10 codes → Random valid diagnosis codes
- ✅ Patient names → Random first/last names
- ✅ Locations → Random cities/addresses

### Processing Pipeline
1. **Regex Pass** - Pattern matching (very fast)
2. **NLP Pass** - Entity recognition using spaCy (intelligent)
3. **Output** - Masked text with structure preserved

### Security
- 🔒 100% offline (no internet needed)
- 🔒 No data persistence (everything in memory)
- 🔒 No external APIs (completely local)
- 🔒 Structure preserved (JSON/SQL intact)
